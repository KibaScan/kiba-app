1. Create scan history service
File: src/services/scanHistoryService.ts
typescriptexport interface ScanHistoryItem {
  id: string;
  product_id: string;
  product_name: string;
  product_brand: string;
  product_image_url: string | null;
  product_form: string;
  product_category: string;
  final_score: number | null;
  is_supplemental: boolean;
  is_recalled: boolean;
  is_vet_diet: boolean;
  bypass_reason: string | null;
  scanned_at: string;
}

export async function getRecentScans(
  petId: string,
  limit?: number,
): Promise<ScanHistoryItem[]>
Query: scans joined with products on product_id, filtered by pet_id, ordered by scanned_at DESC, limited to limit (default 10). Select only the fields needed for the card — don't pull full product data.
If the same product was scanned multiple times, only show the most recent scan (deduplicate by product_id, keep latest scanned_at). Users don't need to see "Blue Buffalo" three times.
2. Create ScanHistoryCard component
File: src/components/ScanHistoryCard.tsx
A compact card for each recent scan. Reuse patterns from PantryCard:

Product image (40×40, rounded, placeholder if no image)
Brand (muted, small) + product name (semibold, 1-line clamp)
Score badge: getScoreColor(score, isSupplemental) → "73% match" in color. For bypasses: show the bypass badge (Recalled / Vet Diet / etc.) same as PantryCard's ScoreBadge.
Relative time: "2h ago", "Yesterday", "3 days ago", "Mar 15" (use a simple helper — no moment.js)
Tap → navigate to ResultScreen with the product (or to RecallDetail if recalled)

Style: Same card background as PantryCard (Colors.card), compact height (~64px), horizontal layout. D-094 compliant — score shows as "X% match", not naked number.

5. Relative time helper
File: src/utils/formatters.ts (add to existing file)
typescriptexport function formatRelativeTime(isoDate: string): string
Logic:

< 1 hour: "Just now"
< 24 hours: "Xh ago"
< 48 hours: "Yesterday"
< 7 days: "X days ago"
Otherwise: "Mar 15" (short month + day)

No external dependency. Pure function, testable.
